Never default to Serra's dark/modern color palette when designing client websites. Every client has a different industry, audience, and brand feel.

**Why:** Josef noticed I kept leaning toward Serra-style themes (dark, tech, blue/purple). This looks wrong for industries like weddings, photography, food, wellness, etc.

**How to apply:** Before designing ANY client site, always:
1. Run a live web research agent for current industry design trends (Awwwards, Behance, Dribbble)
2. Pull colors from their existing site first
3. Match the industry aesthetic — never pull from memory/training data alone
4. Present 3 distinct directions to Josef, let him pick
5. Ask Josef if unsure

**Industry color guides:**
- Wedding/Photography → warm, romantic: cream, champagne, gold, deep burgundy, soft whites
- Restaurant/Food → appetite-driving: warm reds, oranges, earthy browns, forest green
- Wellness/Spa → calm: sage green, lavender, soft beige, muted tones
- Law/Finance → trust: navy, dark grey, white, gold accents
- Tech/SaaS → modern: dark mode, blue, purple (Serra-style is OK HERE only)
- Kids/Education → playful: bright primaries, rounded fonts
- Fitness → energetic: black, red, electric blue, bold typography
