# AI App Ideas for Passive Income

Categorized by difficulty. Start with Beginner tier.

---

## Beginner Tier

These require: Claude API, simple web form.

### 1. Message Summarizer Service
**What:** Users paste a batch of messages/emails → AI returns a clean summary
**Who pays:** Busy professionals, VAs, small business owners
**Price:** $9–$19/month
**Build time:** 1–2 weeks
**Tech:** Claude API + simple web form + Stripe

### 2. Job Application Assistant
**What:** User pastes job description + their resume → gets tailored cover letter + suggestions
**Who pays:** Job seekers (huge market)
**Price:** $5–$15/month or $2/use
**Build time:** 1–2 weeks
**Tech:** Claude API + web form

### 3. Social Media Caption Generator
**What:** Upload a photo or describe a post → get 5 caption options in your style
**Who pays:** Small business owners, content creators
**Price:** $9–$29/month
**Build time:** 1–2 weeks
**Tech:** Claude API + web form

### 4. Meeting Notes Cleaner
**What:** Paste messy meeting notes → get clean, structured summary with action items
**Who pays:** PMs, consultants, remote teams
**Price:** $9–$19/month
**Build time:** 1 week
**Tech:** Claude API + web form

### 5. Niche FAQ Bot
**What:** An AI chatbot trained on a specific topic (e.g., "FAQ bot for Philippines visa questions")
**Who pays:** Website owners in niche industries
**Price:** $29–$99/month per website
**Build time:** 2–3 weeks
**Tech:** Claude API + simple chat widget

---

## Intermediate Tier (After 3–6 months of learning)

These require: web development basics, database knowledge, payment systems.

### 6. Automation Template Packs
**What:** Sell pre-built automation scripts or Claude prompt packs as downloadable products
**Who pays:** Non-technical people who want automation but can't build it
**Price:** $19–$49 per pack
**Build time:** After you've built 10+ automations yourself (Josef already has)
**Tech:** Gumroad or Lemon Squeezy (no-code selling platforms)

### 7. AI Writing Assistant for a Specific Niche
**What:** Claude fine-tuned with prompts for one specific writing type
**Example niches:** Real estate listings, restaurant menus, medical disclaimers, legal summaries
**Who pays:** Professionals in that niche
**Price:** $29–$99/month
**Build time:** 2–4 weeks
**Tech:** Claude API + web app

### 8. Automated Report Generator
**What:** User uploads data (CSV/spreadsheet) → AI generates a narrative report
**Who pays:** Analysts, accountants, small business owners
**Price:** $29–$79/month
**Build time:** 3–4 weeks
**Tech:** Claude API + file upload + web app

### 9. Customer Support Bot Builder
**What:** Let businesses create a custom AI support bot using their own FAQ/docs
**Who pays:** Small e-commerce stores, SaaS companies
**Price:** $49–$199/month
**Build time:** 1–2 months
**Tech:** Claude API + document processing + web interface

---

## Advanced Tier (After 6–12 months of learning)

Only pursue these after completing the beginner tier.

### 10. Vertical AI Agent
**What:** A fully automated AI agent (Claude API + scheduled scripts) for one specific industry workflow
**Example:** "AI bookkeeping assistant for freelancers in the Philippines"
**Who pays:** Businesses wanting automation
**Price:** $99–$499/month
**Build time:** 2–4 months

### 11. White-Label AI Tool
**What:** Build a general AI tool, sell it to agencies who rebrand it for their clients
**Who pays:** Digital marketing agencies
**Price:** $299–$999/month per agency
**Build time:** 3–6 months

---

## Ideas to Avoid (For Now)

- **AI image generation tools** — massively oversaturated, high compute costs
- **General chatbots** — too much competition from free tools
- **Anything requiring real-time data** — complex infrastructure for a beginner
- **AI for regulated industries (medical, legal, financial advice)** — liability issues without expertise
- **Copy of an existing popular tool** — very hard to compete without a distribution advantage

## Philippine Market Opportunities

Specific opportunities given local context:

1. **OFW (Overseas Filipino Worker) document helper** — assist with paperwork, summaries, translations
2. **BPO productivity tools** — call center workers often need summarization/drafting tools
3. **Filipino entrepreneur kit** — AI assistant for DTI registration, basic bookkeeping, customer messaging
4. **English writing assistant for students** — large market, clear willingness to pay
5. **Bahasa/Filipino content tools** — less competition than English-only tools

## How to Validate Any Idea in 48 Hours

1. Post in a relevant Facebook group or Reddit: "Would anyone pay for X? What would you pay?"
2. Build a fake landing page (Carrd.co — free, 30 minutes) describing the product
3. Share the landing page and count clicks on a "Join waitlist" button
4. 10+ signups = worth pursuing. 0–2 signups = rethink.

This costs $0 and takes 2 hours. Do this before writing a single line of code.
